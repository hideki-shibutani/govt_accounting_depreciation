# 日本の公共施設: Japan Local Government Public Facilities Depreciation Data

## 定義
　このリポジトリは、各地方公共団体が公表した「施設類型別ストック情報分析表」を解析し、有形固定資産減価償却率を自動集計して公表するものです。

## 公開データについて
- **ファイル形式**: CSV (UTF-8 with BOM)
- **更新頻度**: 毎月自動更新（データに変更がある場合のみ）

## 収録項目
- 団体コード / 団体名
- 施設類型（facility_id）
- 各年度の有形固定資産減価償却率（3月31日時点）

## データの生成元
　このデータは、[shibutani.org](https://shibutani.org) のデータベースを基に出力されるものです。
